== Description ==

Apply the [Infinite Scroll](http://www.infinite-scroll.com/) plugin to WordPress admin screens, specifically edit.php and edit-comments.php.  No more paginating through the list of posts, pages (including custom post types) or comments!

This is a totally beta plugin, so your mileage may vary.
